📊 Marketing Campaign & Product Analysis
A comprehensive data analysis project focused on measuring marketing effectiveness and customer purchasing behavior. This project bridges the gap between raw data exploration and actionable business intelligence using Python and Power BI.

🎯 Project Objectives
The goal of this project is to evaluate how different marketing campaigns impact sales and to identify key product trends among customers.

EDA: Understand customer demographics and transaction patterns.

Statistics: Validate the impact of campaigns using hypothesis testing.

Data Engineering: Transform and model data for optimal performance.

Visualization: Build a business-ready dashboard to track KPIs.

🛠️ Tech Stack
Analysis: Python (Pandas, NumPy, Matplotlib, Seaborn)

Statistics: Scipy, Statsmodels

BI & ETL: Power BI Desktop, Power Query, DAX

Project Management: Jira, Git, GitHub

🚀 Project Workflow
1. Exploratory Data Analysis (Python)
Identified variable types and handled missing values/anomalies.

Generated descriptive statistics (mean, median, standard deviation).

Visualizations: Used boxplots and histograms to detect outliers in spending and bar charts to compare campaign acceptance rates.

2. Statistical Analysis
Impact Testing: Conducted t-tests and ANOVA to determine if campaign participants spent significantly more than non-participants.

Correlation: Used Chi-square tests to analyze relationships between categorical variables like "Country" and "Campaign Acceptance."

Interpretation: Filtered insights based on p-values to ensure findings were statistically significant.

3. Data Transformation & Modeling (Power Query/BI)
ETL: Pivoted/Unpivoted data to create normalized tables for Campaigns, Products, and Platforms.

Data Modeling: Established a Star Schema using Customer ID as the primary key.

DAX Measures: Developed custom measures for total revenue, campaign conversion rates, and platform distribution.

4. Interactive Dashboard
The final Power BI report focuses on Campaign Performance, featuring:

Revenue Analysis: Breakdown of sales by product and campaign.

Channel Insights: Distribution of purchases across Web, Store, and Catalog.

Dynamic Filtering: Slicers for demographics and timeframes to allow for deep-dive analysis.

📂 Deliverables
Python Notebook: Contains the full EDA process and statistical test interpretations.

Power BI File (.pbix): The interactive dashboard and data model.
